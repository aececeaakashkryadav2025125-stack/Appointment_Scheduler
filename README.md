# 🚀 Appointment Booking System

## 📌 Project Overview
A full-stack appointment booking system where users can register, login, and book appointments with time-slot conflict prevention.

---

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP (REST API)
- Database: MySQL
- Deployment: 
  - Frontend → Vercel
  - Backend → PHP Hosting

---

## ⚙️ Setup Instructions

### 🔹 Backend Setup
1. Install XAMPP
2. Start Apache & MySQL
3. Import database in phpMyAdmin
4. Place backend folder inside `htdocs`
5. Run:
   http://localhost/appointment-system/backend/index.php

---

### 🔹 Frontend Setup
1. Open:
   http://localhost/appointment-system/frontend/index.html
